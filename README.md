# GraphQL-Yoga + Graphql Tools

`pnpm init`
`pnpm add express graphql graphql-yoga`

https://the-guild.dev/graphql/tools/docs/schema-loading
`pnpm add @graphql-tools/load-files`

`node index`

`pnpm add -D nodemon`

Add this in package.json
`"start": "nodemon index.js",`
`pnpm start`
